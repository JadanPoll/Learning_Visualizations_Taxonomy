2. Calculating Redundancy

"The other three are redundant... chosen to make sums EVEN:"
X1 = (X3 + X5 + X7) mod 2
   = (1+0+0) mod 2 = 1

X2 = (X3 + X6 + X7) mod 2
   = (1+1+0) mod 2 = 0

X4 = (X5 + X6 + X7) mod 2
   = (0+1+0) mod 2 = 1

Receiver: Syndrome Calculation

"When a block of seven is received α, β and γ are calculated..."
γ = (R1+R3+R5+R7) mod 2
  = (1+1+0+0) mod 2 = 0

β = (R2+R3+R6+R7) mod 2
  = (0+1+1+0) mod 2 = 0

α = (R4+R5+R6+R7) mod 2
  = (1+0+1+0) mod 2 = 0
