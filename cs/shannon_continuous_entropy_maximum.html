Continuous Entropy (Sec. 20)

Average Signal Power

Set the fixed Variance (σ2). Both distributions below will be forced to contain this exact amount of power (spread).
Variance (σ2) 4.00

Gaussian Distribution

H = ∫ -p(x) log2 p(x) dx
HG = log2(√2πe · σ2)
HG = log2(√17.079 · 4.00)
HG = 3.047 bits

Uniform Distribution

H = ∫ -p(x) log2 p(x) dx
HU = log2(√12 · σ2)
HU = log2(√12 · 4.00)
HU = 2.792 bits
Conclusion: For any given power σ2, the Gaussian distribution always carries exactly 0.254 bits more entropy than a Uniform distribution. It is the absolute maximum entropy shape.
