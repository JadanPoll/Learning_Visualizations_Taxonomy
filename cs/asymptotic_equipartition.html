Theorems 3 & 4 (The AEP)

"Given any ε > 0 and δ > 0, we can find an N0 such that the sequences of any length N ≥ N0 fall into two classes... [The main class] all of whose members have probabilities satisfying the inequality: | (log p-1)/N - H | < δ"

This means that for large N, almost all sequences that actually occur have the exact same probability: 2-NH. The number of these sequences is exactly 2NH.
