Symbol Capacity (Sec. 1)

Symbol Durations

Set the time (in arbitrary units) it takes to transmit each symbol. Example: Telegraphy (Dot = 1, Dash = 3).
Symbol A (t1) 1 unit
Symbol B (t2) 3 units

Characteristic Equation

Find the real root X0 where the sum of X-t equals 1.
f(X) = X-t1 + X-t2
1 = X-1 + X-3
→ X0 = 1.4656

Maximum Channel Capacity (C)

The absolute limit of bits per unit time this system can transmit.
C = log2(X0)
C = log2(1.4656)
C = 0.551 bits/unit
